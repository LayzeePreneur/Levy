## Features

- Save daily expenses and spendings
- Show spending chart for recent 7 days
- Show alert dialogue when deleting expense
- Landscape mode support
- Cross platform support
- Good looking ui
- Attractive design
- High performance
- Smooth scrolling and performance
- Custom Google fonts
